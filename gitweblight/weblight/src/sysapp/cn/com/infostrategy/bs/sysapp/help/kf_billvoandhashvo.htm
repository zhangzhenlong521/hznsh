HashVO:
1、简述:
简单理解可以看成一个带排序的HashMap。此类常用作查询数据库时返回的数据对象。
2、常用方法介绍：
public String getStringValue(String attrname, String _isNullConvertStr)
public int getIntegerValue(String attrname, int _nvl)
public Long getLognValue(String attrname, long _nvl)
public Double getDoubleValue(String attrname, Double _nvldouble)
public Boolean getBooleanValue(String attrname, boolean _nvl)
public java.sql.Date getDateValue(String attrname)
public java.sql.Timestamp getTimeStampValue(String attrname)
以上为常用获取某一项的值的常用方法第2个参数为如果取到的值为null时的默认值
public void setAttributeValue(String key, Object value)//设置某一项的值
public void removeItem(String _key)//移除某一项
public HashVO clone()//克隆
public HashVO deepClone()//克隆

BillVO:
1、简述:
简单理解可以看成由几个数组和一些变量组成的数据对象。主要有存实际数据的数组、存key的数组、
存name的数组、存控件类型的数组、模板名、保存查询的表名。其中实际数据存储的是经过封装的
VO(StringItemVO、RefItemVO、ComBoxItemVO)。BillVO一般可以从UI端的卡片列表以及树中得到。
2、方法介绍：
public Object getObject(String _key)//获取某一项的实际VO需要类型转化一下
public String getRealValue(String _key)//获取某一项的实际值
public String getStringValue(String _key, String _nvl)//获取某一项的实际值
public String getStringViewValue(String _key)//获取某一项的显示值
public void setObject(String _key, Object _obj)//设置某一项的VO
public String getInsertSQL()//通过billvo获取插入sql
public String getUpdateSQL()//通过billvo获取更新sql
public String getDeleteSQL()//通过billvo获取删除sql
public HashVO convertToHashVO()//将billvo的实际值转化成hashvo
public HashVO convertViewToHashVO()//将billvo的显示值转化成hashvo
public String getTempletName()//获取模板名称
public String getSaveTableName()//获取保存表名
public String getQueryTableName()//获取查询表名
